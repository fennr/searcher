# PROP-011: Reasoning по умолчанию и short-режим

## Статус
Принято

## Контекст
Для повседневного использования полезнее сначала видеть краткое объяснение и 1-3 вероятных команды. Командный сценарий с немедленным запуском должен оставаться доступным, но как явный opt-in режим.

## Решение
- Сделать reasoning-режим поведением по умолчанию.
- Ввести короткий режим `-s`/`--short` для сценария выбора и запуска одной команды.
- В short-режиме добавить явную отмену выбора командой `0` (в дополнение к Enter).
- Уточнить prompt: модель должна давать 1-3 наиболее вероятные shell-команды.

## Последствия
- Плюсы: дефолтный вывод становится более информативным и безопасным.
- Плюсы: short-режим остается быстрым и управляемым для прямого запуска.
- Минусы: изменяется привычный CLI-контракт (флаг `-r` больше не основной сценарий).
