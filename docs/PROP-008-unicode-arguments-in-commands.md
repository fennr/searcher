# PROP-008: Поддержка Unicode-аргументов в командах

## Статус
Принято

## Контекст
Валидация команд отклоняла любые строки с кириллицей, из-за чего корректные команды с Unicode-аргументами считались невалидными.

## Решение
Изменить эвристику валидации:

- Требовать корректный первый исполняемый токен команды.
- Разрешить Unicode в аргументах команды.
- Запрещать управляющие символы.

## Последствия

- Плюсы: корректно обрабатываются команды вида `grep -i "требования"` и аналогичные.
- Плюсы: меньше ложных ошибок при локализации запросов.
- Минусы: эвристика становится чуть менее строгой по чистому ASCII.
