# AGENTS

Реализация любого нового функционала в проекте выполняется строго по следующей логике:

1. Чтение всех предшествующих архитектурных решений из `docs/PROP-*`.
2. Добавление нового документа с описанием нового предложения и его последствий.
3. Реализация нового предложения в коде после фиксации решения в документации.
